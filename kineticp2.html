Adaptive Façade System

This project aimed to develop an adaptive façade system based on the logic of Kaleidocycle origami. Programmed using Arduino, the façade rotates between 0 and 100 degrees, transitioning from semi-closed to fully closed views. This dynamic movement provides responsive control over light, privacy, and ventilation.
